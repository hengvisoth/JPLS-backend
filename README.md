## Start server
* run ```mongod``` or ```sudo mongod``` on cmd/terminal
* ```npm start```
### Requirements

##### Install yarn
* ```npm i -g yarn``` or ```sudo npm i -g yarn```
##### Install npm modules
* ```yarn install```
#### MongoDB
* download https://docs.mongodb.com/manual/administration/install-community/
